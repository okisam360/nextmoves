# NextMove

Este es el repositorio del proyecto NextMove, un sitio web basado en WordPress.

## Estructura del Proyecto

- `wp-admin/`: Archivos del núcleo de administración de WordPress.
- `wp-content/`: Temas, plugins y subidas.
- `wp-includes/`: Archivos del sistema de WordPress.
- `docs/`: Documentación adicional del proyecto.

## Documentación Relevante

- [ACCESOS.md](ACCESOS.md): Información de acceso FTP y Panel.
- [PROJECT_RULES.md](PROJECT_RULES.md): Reglas específicas del proyecto.
- [docs/COLOR.md](docs/COLOR.md): Paleta de colores del diseño.
- [docs/TIPOGRAFIAS.md](docs/TIPOGRAFIAS.md): Tipografías utilizadas.

## Configuración Local

Para trabajar en este proyecto localmente, asegúrate de tener un entorno PHP y MySQL configurado (como LocalWP, XAMPP o Docker).

1. Clona el repositorio.
2. Configura tu archivo `wp-config.php` con las credenciales locales.
3. Importa la base de datos (si está disponible).
